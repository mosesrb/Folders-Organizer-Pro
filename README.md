# Folders Organizer Pro (v4)

A professional desktop utility for managing digital workspace entropy. Built with a high-performance Python backend and a pixel-perfect React frontend.

## 🚀 Quick Start

1. **Prerequisites**
   - Python 3.10+
   - Node.js (for frontend development)

2. **Setup**
   ```bash
   # Install backend dependencies
   pip install pywebview

   # Setup frontend
   cd ui
   npm install
   npm run build
   ```

3. **Run**
   ```bash
   python organizer.py
   ```
